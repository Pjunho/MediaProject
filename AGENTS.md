# AGENTS.md

## 기본 규칙
- 이 프로젝트에서는 기본적으로 한국어로 답변한다.
- Unity/C# 코드 변경 전 관련 파일을 먼저 확인한다.
- 기존 구조와 스타일을 우선 따르고, 요청 범위를 벗어난 리팩터링은 피한다.
- UI 문구, 정렬, 클릭 동작, 한국어 글꼴/표시 안정성을 중요한 요구사항으로 본다.

## 검증
- 코드 변경 후 가능한 범위에서 Unity 콘솔, 스크립트 검증, 관련 실행 경로를 확인한다.
- Unity refresh나 컴파일 확인이 실패하면 실패 사실과 대체 확인 결과를 함께 남긴다.

## Git 규칙
- 파일 생성, 수정, 삭제를 완료하면 `git add`와 `git commit`을 수행한다.
- 요청 범위 밖의 기존 삭제/미추적 파일은 상태 확인, 언급, 스테이징, 커밋 대상으로 삼지 않는다.
- 커밋 메시지는 한국어로 간결하게 작성한다.
- 원격 저장소: https://github.com/Pjunho/MediaProject
