# 오디오 크레딧

이 프로젝트에 포함된 BGM은 모두 OpenGameArt.org의 CC0 라이선스 음원입니다.
CC0라서 사용·수정·상업 배포가 가능하며 크레딧 표기는 필수는 아니지만, 출처 추적을 위해 아래에 남깁니다.

## 효과음

버튼 클릭, 출전 확정, 웨이브 클리어/실패 효과음은 `Assets/GameAudio.cs`에서 런타임에 직접 생성합니다.
별도 외부 음원 파일이나 추가 라이선스 표기가 필요하지 않습니다.
