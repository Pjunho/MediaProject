# 랜덤 오펜스 - 게임 기획 인덱스

이 파일은 자동/공통 컨텍스트에서 가볍게 읽기 위한 요약 인덱스다.
세부 규칙은 필요한 항목의 하위 문서만 열어본다.

## 핵심 루프

적과 맵이 매 스테이지마다 랜덤으로 생성되는 전략 게임.
플레이어는 웨이브마다 아군 경로와 출전 순서, 출전 타이밍을 조정하여 목표까지의 도달을 노린다.

## 공통 불변 규칙

- 스테이지당 15웨이브
- 5/10/15웨이브 클리어 시 별 1/2/3개 획득
- 별 3개 획득 시 해당 스테이지 보석 획득
- 아군 스킬은 스테이지 시작 시 잠금 상태이며, 진행 중 코인으로 해금
- 아군 속도/체력 업그레이드는 각 3단계, 비용은 1/3/6코인
- 실패 시 해당 스테이지 처음부터 재시작
