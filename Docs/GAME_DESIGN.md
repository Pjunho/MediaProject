# 랜덤 오펜스 - 게임 기획 문서

## 게임 개요

적과 맵이 매 스테이지마다 랜덤으로 생성되는 전략 게임.
플레이어는 웨이브마다 아군의 경로와 조합을 수정하며 골에 도달시키는 것이 목표.

## 코인 시스템

- 웨이브 클리어 시 골에 도달한 아군 수만큼 1코인씩 획득
- 아군이 골에 도착하는 즉시 코인이 지급된다.
- 스테이지 시작 시 기본 코인 지급
- 코인 용도:
  - 아군 해금
  - 스킬 해금
  - 아군 스탯 업그레이드

## 스킬 시스템

- 아군의 스킬은 스테이지 시작 시 잠겨 있음
- 스테이지 진행 중 코인으로 해금 가능
- 스테이지 클리어 또는 실패 후 재시작하면 다시 잠김
- 스킬 아이콘은 `Assets/Resources/Icon`에 위치

### 아군별 스킬 방향

- 전사: 불굴의 의지. 2초간 모든 공격 피해 무시
- 궁수: 마비 화살. 대상 지정 전 게임을 0.1배속으로 느리게 진행하고, 클릭한 적을 3초간 공격 불가 상태로 만듦
- 마법사: 순간 보호막. 1.5초간 일정 범위 내 아군을 파란 보호막으로 보호
- 성직자: 치유의 빛. 생존 아군 체력을 5초간 초당 3% 회복, 회복 대상은 노란빛 표시
- 도적: 연막탄. 생존 아군에게 5초간 적 공격 명중 확률 50% 감소 효과, 보라색 연기 표시
- 성기사: 수호의 맹세. 4초간 아군 피해의 80%를 대신 받고, 자신은 초당 5% 회복. 대상 아군은 피격 경직을 받지 않고 성기사가 대신 경직됨

> 참고: `SkillSystem.cs`의 스킬명과 효과는 사용자가 별도 세션에서 직접 수정 완료했음. 자동 작업 시 스킬명 불일치를 이유로 재수정하지 않는다.

## 아군 업그레이드 시스템

- 코인으로 아군의 속도와 체력을 각각 3단계까지 업그레이드 가능
- 기본 속도/체력은 아군마다 상이
- 업그레이드 비용:
  - 1단계: 1골드
  - 2단계: 3골드
  - 3단계: 6골드

## 스테이지 구성

### 기본 원칙

- 앞 스테이지일수록 쉽고, 뒤 스테이지일수록 어려움
- 앞 스테이지 클리어로 얻은 보석이 뒤 스테이지 공략에 도움이 되도록 설계
- 각 맵마다 5웨이브 클리어 시 별 1개, 10웨이브 클리어 시 별 2개, 15웨이브 클리어 시 별 3개 획득
- 별 3개까지 획득해야 보석을 획득할 수 있음

### 스테이지 목록

| 스테이지 | 테마 | 적 구성 |
| --- | --- | --- |
| 1 | 초원의 전투 | 근접 적 다수, 원거리 적 소수 |
| 2 | 사막의 요새 | 근접 / 원거리 비슷한 비율 |
| 3 | 화산의 심판 | 근접 적 소수, 원거리 적 다수 |

### 출전 가능 아군 수

- 1스테이지: 4명
- 2스테이지: 5명
- 3스테이지: 6명

## 웨이브 구성

스테이지당 15웨이브.

| 구간 | 맵 크기 | 적 수/강도 | 아군 수 | 최단 루트 난이도 |
| --- | --- | --- | --- | --- |
| 1~5 웨이브 | 소 | 적음, 기본 | 기본 | 낮음 |
| 6~10 웨이브 | 중 | 다소 증가, 강한 적 추가 | +1명 | 중간 |
| 11~15 웨이브 | 대 | 많이 증가, 강력한 적 추가 | +2명 | 높음 |

### 랜덤성 설계 의도

- 적은 랜덤 스폰이므로 1웨이브 배치에 따라 난이도가 달라짐
- 유리한 배치를 원하면 스테이지를 처음부터 재시작해야 함

## 보석 시스템

- 스테이지를 별 3개까지 클리어 시 해당 스테이지 고유의 보석 1개 획득
- 보석은 영구 버프 효과를 가지며 이후 스테이지 공략에 활용 가능
- 클리어 시 자동으로 활성화됨
- 메인 메뉴 내 보석 메뉴에서 확인 가능
- 보석별 활성화 / 비활성화 토글 가능

| 스테이지 | 보석 이름 | 보석 효과 |
| --- | --- | --- |
| 1 초원의 전투 | 초원의 보석 | 아군 이동 속도 +15% |
| 2 사막의 요새 | 사막의 보석 | 아군 최대 HP +20% |
| 3 화산의 심판 | 화산의 보석 | 아군 이동 속도 +10%, 최대 HP +10% |

## 보너스 코인

- 맵에 최대 2개까지 코인이 발생할 수 있음
- 위치는 랜덤
- 각 웨이브마다 등장 확률은 5%

## 수동 출전

- 아군 출전 순서를 정한 후 스타트 버튼을 누름
- 이후 스페이스바 또는 클릭으로 순서대로 아군을 1명씩 스폰

## 게임 종료 조건

- 15웨이브 전부 클리어: 스테이지 클리어
- 실패 시 해당 스테이지 처음부터 재시작

## 완료된 주요 결정

- 스킬 기본 해금 여부: 잠금 상태로 시작, 코인으로 해금
- 스테이지 2, 3 테마명 및 보석 효과 확정
- 스테이지별 기본 코인 지급량: 1:3, 2:5, 3:7, 4:10, 5:12
- 스테이지별 아군 초기 배치 수: 1:4, 2:5, 3~5:6명
- 웨이브 수: 15웨이브

- 스킬 효과 시각화 구현 완료
- 웨이브 HUD 별 진행 표시기 구현 완료
- 골 도달 시 플로팅 텍스트 효과 구현 완료
