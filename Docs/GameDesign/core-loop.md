# 핵심 루프

## 게임 개요

적과 맵이 매 스테이지마다 랜덤으로 생성되는 전략 게임.
플레이어는 웨이브마다 아군의 경로와 조합을 수정하며 골에 도달시키는 것이 목표.

## 웨이브 진행

- 아군 출전 순서를 정한 후 스타트 버튼을 누름
- 이후 스페이스바 또는 클릭으로 순서대로 아군을 1명씩 스폰
- 적은 랜덤 스폰이므로 1웨이브 배치에 따라 난이도가 달라짐
- 유리한 배치를 원하면 스테이지를 처음부터 재시작해야 함

## 게임 종료 조건

- 15웨이브 전부 클리어: 스테이지 클리어
- 실패 시 해당 스테이지 처음부터 재시작
