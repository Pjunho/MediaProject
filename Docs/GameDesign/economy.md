# 경제 시스템

## 코인

- 웨이브 클리어 시 골에 도달한 아군 수만큼 1코인씩 획득
- 아군이 골에 도착하는 즉시 코인이 지급된다
- 스테이지 시작 시 기본 코인 지급
- 코인 용도:
  - 아군 해금
  - 스킬 해금
  - 아군 스탯 업그레이드

## 보너스 코인

- 맵에 최대 2개까지 코인이 발생할 수 있음
- 위치는 랜덤
- 각 웨이브마다 등장 확률은 5%

## 업그레이드 비용

- 코인으로 아군의 속도와 체력을 각각 3단계까지 업그레이드 가능
- 기본 속도/체력은 아군마다 상이

| 단계 | 비용 |
| --- | --- |
| 1단계 | 1코인 |
| 2단계 | 3코인 |
| 3단계 | 6코인 |
