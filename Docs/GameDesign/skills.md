# 스킬 시스템

## 기본 규칙

- 아군의 스킬은 스테이지 시작 시 잠겨 있음
- 스테이지 진행 중 코인으로 해금 가능
- 스테이지 클리어 또는 실패 후 재시작하면 다시 잠김
- 스킬 아이콘은 `Assets/Resources/Icon`에 위치

> 참고: `SkillSystem.cs`의 스킬명과 효과는 사용자가 별도 세션에서 직접 수정 완료했음. 자동 작업 시 스킬명 불일치를 이유로 재수정하지 않는다.

## 아군별 스킬 방향

| 아군 | 스킬 | 효과 |
| --- | --- | --- |
| 전사 | 불굴의 의지 | 2초간 모든 공격 피해 무시 |
| 궁수 | 마비 화살 | 대상 지정 전 0.1배속 진행, 클릭한 적을 3초간 공격 불가 상태로 만듦 |
| 마법사 | 순간 보호막 | 1.5초간 일정 범위 내 아군을 파란 보호막으로 보호 |
| 성직자 | 치유의 빛 | 생존 아군 체력을 5초간 초당 3% 회복, 회복 대상은 노란빛 표시 |
| 도적 | 연막탄 | 생존 아군에게 5초간 적 공격 명중 확률 50% 감소 효과, 보라색 연기 표시 |
| 성기사 | 수호의 맹세 | 4초간 아군 피해의 80%를 대신 받고 초당 5% 회복. 대상 아군은 피격 경직을 받지 않고 성기사가 대신 경직됨 |
