# 역디펜스 게임 (랜덤 디펜스)

아주대학교 미디어학과 4학년 1학기 미디어프로젝트(종합설계) 과제로 개발한 Unity 전략 게임입니다.

---

## 저장소 소개

이 저장소는 Unity 기반 역디펜스 장르의 게임 프로젝트입니다.  
일반적인 타워 디펜스와 달리, **플레이어가 직접 아군 유닛을 조종하여 적진을 돌파**하는 구조입니다.  
매 스테이지마다 맵과 적 배치가 랜덤으로 생성되어 매번 다른 전략이 요구됩니다.

---

## 프로젝트 개요

- **장르**: 역디펜스 / 전략
- **핵심 루프**: 랜덤 생성된 맵에서 아군 경로와 출전 순서를 조정하며 골 도달을 목표로 함
- **스테이지**: 총 3개 (초원의 전투 → 어둠의 동굴 → 화산의 심판)
- **웨이브**: 스테이지당 15웨이브, 5웨이브 단위로 난이도 상승
- **승리 조건**: 지정된 수 이상의 아군이 골 지점에 도달

### 주요 시스템

| 시스템 | 설명 |
|--------|------|
| 경로 설정 | 웨이브 시작 전 아군이 이동할 경로를 직접 그림 |
| 출전 순서 | 아군 6종 중 원하는 조합과 순서를 편성 |
| 스킬 | 아군별 고유 스킬, 코인으로 해금 후 수동 발동 |
| 업그레이드 | 아군 속도/체력을 코인으로 3단계까지 강화 |
| 보석 | 스테이지 별 3개 클리어 시 획득, 영구 버프 효과 |
| 보너스 코인 | 경로 위에 랜덤 배치, 아군이 지나가며 획득 |

### 아군 종류 및 스킬

| 아군 | 스킬 | 효과 |
|------|------|------|
| 전사 | 불굴의 의지 | 2초간 모든 피해 무시 |
| 궁수 | 마비 화살 | 지정한 적을 3초간 공격 불가 상태로 만듦 |
| 마법사 | 순간 보호막 | 1.5초간 범위 내 아군 보호 |
| 성직자 | 치유의 빛 | 5초간 생존 아군 체력 초당 3% 회복 |
| 도적 | 연막탄 | 5초간 적 명중률 50% 감소 |
| 성기사 | 수호의 맹세 | 4초간 아군 피해의 80%를 대신 받고 초당 5% 회복 |

### 보석 목록

| 스테이지 | 보석 | 효과 |
|----------|------|------|
| 1 초원의 전투 | 초원의 보석 | 아군 이동 속도 +15% |
| 2 어둠의 동굴 | 사막의 보석 | 아군 최대 HP +20% |
| 3 화산의 심판 | 화산의 보석 | 획득 코인 +50% |

## 빌드 및 실행 안내

### 실행 방법
1. 이 저장소를 클론합니다.
   ```
   git clone https://github.com/Pjunho/MediaProject.git
   ```
2. Unity Hub에서 **Unity 6000.3.x** 버전으로 프로젝트를 엽니다.
3. `Assets/Scenes` 폴더에서 메인 씬을 열고 Play 버튼으로 실행합니다.

### 주의 사항
- Unity 버전이 다를 경우 일부 URP 설정이나 씬 파일이 정상적으로 열리지 않을 수 있습니다.
- `Library/`, `Temp/`, `Build/` 폴더는 `.gitignore`에 의해 저장소에 포함되지 않으므로, 처음 열 때 Unity가 자동으로 생성합니다.
